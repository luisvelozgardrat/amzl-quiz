# 🎮 Amazon Hub Trainer

## ¿Qué es?
App gamificada para estudiar las métricas del DBR de AMZL HUB Location. Incluye:
- 4 modos de juego (Arcade, Supervivencia, Blitz, Estudio)
- 137+ preguntas sobre métricas, operaciones, finanzas y más
- Escenarios reales del DBR
- Glosario con 120+ términos y pronunciación en inglés
- Sistema de XP, logros y racha diaria
- Modo Podcast con explicaciones
